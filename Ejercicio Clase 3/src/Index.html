En el siguiente articulo se explica que es y como utilizar la etiqueta <template> y las posibilidades que esta brinda

La etiqueta <template> es utilizada para guardar contenido html y luego reutilizarlo multiples veces en nuestra aplicacion, de manera similar a como hacen motores de templating como handlebars y hbs o como funcionan los componentes en librerias mas avanzadas como react.

Para utilizar la etiqueta, primero deberemos generar el codigo html que se desea componetizar encerrado entre la etiquetea <template></template>. Esto generara el codigo html del componente, pero todavia no sera visible para el usuario.
Para poder hacer el componente visible al usuario, debemos hacer uso de javascript. Primero debermos generar un wrapper en nuestro html en donde insertar nuestra template como hijo, y luego utilizar un codigo javascript parecido al siguiente. 
<script>
   let myArr = ["Audi", "BMW", "Ford", "Honda", "Jaguar", "Nissan"];
   let temp, item, a, i;
   temp = document.getElementsByTagName("template")[0];
   item = temp.content.querySelector("div");
   for (i = 0; i < myArr.length; i++) {
      a = document.importNode(item, true);
      a.textContent += myArr[i];
      document.body.appendChild(a);
   }
</script>
En el codigo de arriba, vemos como utilizando javascript podemos tomar un template y injectarlo en nuestro html (en este caso en le body) multiples veces cambiando o no los valores del mismo.
